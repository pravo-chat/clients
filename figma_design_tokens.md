### Примечание о несоответствии
- В meta.json указан фон: RGB(30, 30, 30)
- В коде используется: BackgroundMain = "#F9F9FA" (светло-серый)

## Рекомендации

Для полного извлечения дизайн-токенов из Figma:
1. Использовать Figma API (требуется Personal Access Token)
2. Настроить MCP Figma для постоянного доступа
3. Экспортировать дизайн-спецификации через Figma Dev Mode
